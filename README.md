## Description
AGD is a modern web application inspired by an online manga store, built with Angular. It allows users to browse a wide collection of manga, explore different genres, view detailed information about each title, and manage their reading or shopping experience through a clean and user-friendly interface.

The application simulates a basic manga shop, providing essential features such as manga listings, detailed pages, and smooth navigation using Angular routing.

## Features
- Browse a list of manga titles  
- View detailed information about each manga (description, genre, etc.)  
- Clean and intuitive user interface  
- Fast navigation using Angular routing
