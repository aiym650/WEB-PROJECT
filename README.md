## Description
AGD is a modern web application for an online store built with Angular.  
The platform allows users to browse products, view detailed information, and manage their shopping experience in a simple and user-friendly interface.

The application is designed to simulate a real e-commerce system, providing essential features such as product listing, product details, and navigation between pages using Angular routing.

## Features
- View a list of products
- See detailed information about each product
- Simple and clean user interface
- Fast navigation using Angular routing
