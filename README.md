## Description
AGD is a modern web application inspired by a mini Spotify-like platform, built with Angular. It allows users to browse music content, explore tracks, view detailed information, and manage their listening experience through a clean and user-friendly interface.

The application simulates a basic music streaming service, providing essential features such as track listings, detailed views, and smooth navigation between pages using Angular routing.

## Features
-Browse a list of songs or music items
-View detailed information about each track
-Simple and clean user interface
-Fast navigation using Angular routing
